just say data

Here you can find formulas for: Mean, Median, Mode, Standard Deviation, Skewness coefficient, Amplitude, Quartile, etc.


Mean:

Ungrouped data:
\[ \bar{x} = \frac{1}{n} \sum_{i=1}^{n} x_i \]

Grouped data:
\[ \bar{x} = \frac{\sum f_i x_i}{\sum f_i} \]

\( f_i \) => frequency of the interval
\( x_i \) => median value of the interval

Median:

Ungrouped data:
\[ Me(X) = \begin{cases} x_{\frac{n+1}{2}} & n \text{ odd} \\ \frac{x_{\frac{n}{2}} + x_{\frac{n}{2}+1}}{2} & n \text{ even} \end{cases} \]

the value that stands in the middle of the sorted X; otherwise the mean of the 2 values sharing the midfield.

Grouped data:
\[ Me(X) = l + \frac{\frac{n}{2} - f_0}{f_1} \, h \]

\( l \) => minimum of median class/interval
\( f_0 \) => cumulative frequency of the class
\( f_1 \) => frequency of the class
\( h \) -> interval's width

Mode:

Ungrouped data:
\[ Mo(X) = \arg\max_x \; \text{freq}(x) \]

the value of X that repeats most often; there could be 1, 2, or at most 3 modal values.

Grouped data:
\[ Mo(X) = l + \frac{f_1 - f_0}{2 f_1 - f_0 - f_2} \, h \]

\( l \) => minimum of the modal class/interval
\( f_0 \) => frequency of the preceeding class
\( f_1 \) => frequency of the modal class
\( f_2 \) => frequency of the succeding class
\( h \) -> interval's width

Standard Deviation:

Ungrouped data:
Variance
\[ \sigma^2 = \frac{1}{n} \sum_{i=1}^{n} (x_i - \mu)^2 \]
Standard Deviation
\[ \sigma = \sqrt{\frac{1}{n} \sum_{i=1}^{n} (x_i - \mu)^2} \]

\( \mu \) => mean value of X

standard deviation is the square root of variance; telling you how far values typically lie from the mean (since variance is obtained by squared distances from the mean).

Grouped data:
Variance
\[ \sigma^2 = \frac{\sum f_i (x_i - \bar{x})^2}{\sum f_i} \]
Standard Deviation
\[ \sigma = \sqrt{\frac{\sum f_i (x_i - \bar{x})^2}{\sum f_i}} \]

\( x_i \) => median value of the \( i^{th}\) class/interval
\( \bar{x} \) => mean value of the sample
\( f_i \) => frequency of the \( i^{th}\) class

Skewness coefficient:

Ungrouped data:
\[ \text{Sk} = \frac{3\bigl(\mu - Me(X)\bigr)}{\sigma} \]

\( \mu \) => mean value of X

positive value indicates skewness from the right.

Grouped data:
\[ \text{Sk} = \frac{3\bigl(\bar{x} - Me(X)\bigr)}{\sigma} \]

\( \bar{x} \) => mean value of the sample

Amplitude:

Ungrouped data:
\[ \text{Amp} = x_{max} - x_{min} \]

too sensitive

Grouped data:
\[ \text{Amp} = l_{max} - l_{min} \]

\( l_{max} \) => highest value of the highest class
\( l_{min} \) => lowest value of the lowest class

Quartiles:

Ungrouped data:
\( Q_1 = x_{\frac{n+1}{4}} \)
\( Q_2 = Me(X) \)
\( Q_3 = x_{\frac{3(n+1)}{4}} \)

\( IQR = Q_3 - Q_1 \)

Grouped data:
\[ Q_k = l + \frac{k\frac{n}{4} - f_0}{f_1} \, h \quad (k=1,2,3) \]

\( l \) => lowest value of the class containing the \(Q_k\) value
\( f_0 \) => cumulative frequency of the class
\( f_1 \) => frequency of the class

Permutation && Combinatorics

Here you can find formulas for: Permutation with repetition, permutation without repetition, combinations, etc.


Permutation with repetition:

All data:
\[ P(n,n) = n^{n} \]

Order matters, elements can be repeated.

Selection r:
\[ P(n,r) = n^{r} \]

\( n \) => all available choices
\( r \) => selection size (e.g. places to fill)

Permutation without repetition:

All data:
\[ P(n,n) = n! \]

Order matters, elements can be chosen 1 time.

Selection r:
\[ P(n,r) = \frac{n!}{(n-r)!} \]

\( n \) => all available choices
\( r \) => selection size (e.g. places to fill)

Permutation with certain identical items:

All data:
\[ P(n,n) = \frac{n!}{p_1!p_2!...p_k!} \]

Order matters, identical items are counted only once.

Selection r:
\[ P(n,r) = \frac{n!}{(n-r)!p_1!p_2!...p_k!} \]

\( n \) => all available choices
\( r \) => selection size (e.g. places to fill)
\( p_k \) => identical items

Combination:

All data:
\[ C(n,n) = \binom{n}{n} = 1 \]

Order doesn't matter.

Selection r:
\[ C(n,r) = \binom{n}{r} = \frac{n!}{(r!)(n-r)!} \]

\( n \) => all available choices
\( r \) => selection size (e.g. places to fill)

Properties:

\[ P(n,r) = C(n,r)*r! \]
\[ C(n,0) = \binom{n}{0} = 1 \]

Probability axioms

This page took this project beyond the thousand line barrier.

Below you can see definitions/theorems for Set Theory, Probability Axioms and anything in between, including Bayes', chain rule, etc.


Set Theory:

Union: (OR) \( A \cup B \)
Either A or B happens, or both.
Intersection: (AND) \( A \cap B \)
Both A and B happen at the same time.
Complement: (NOT) \( A^C \) or \( \overline A \)
Event A doesn't happen.
Difference: (A but not B) \( A - B \)
A happens, B doesn't.

Properties:

Union: \[ A \cup (B \cup C) = (A \cup B) \cup C \] \[ A \cup B = B \cup A \] \[ A \cup A = A \] \[ A \cup \Omega = \Omega \] Intersection: \[ A \cap (B \cap C)=(A \cap B) \cap C \] \[ A \cap B = B \cap A \] \[ A \cap A = A \] \[ A \cap \Omega = A \] Complement: \[ \overline{A}= \Omega - A \] Disjoint events (Mutually Exclusive): \[ A \cap B = \empty \] Symmetric difference: \[ A \Delta B = (A-B) \cup (B-A) \]

\( \Omega \) => Sample Space
\( \omega \) => Sample Outcome

De morgan's law:

\[ \overline{A \cup B}= \overline{A} \cap \overline{B} \] \[ \overline{A \cap B}= \overline{A} \cup \overline{B} \]

Number of Elements:

\[ n(A \cup B)= n(A) + n(B) - n(A \cap B) \] Disjoint sets: \[ n(A \cup B)=n(A) + n(B) \] Three sets: \[ n(A \cup B \cup C) = n(A) + n(B) + n(C) \\ - n(A \cap B) - n(A \cap C) - n(B \cap C) \\ + n(A \cap B \cap C) \] Complemets: \[ n(\overline{A})=n( \Omega) - n(A) \] \[ n(\overline{A} \cap \overline{B})=n(\Omega) - n(A \cup B) \]

Partition:

If sets \(A_1, A_2, ...,A_n\) are a partition of \(\Omega\): \( A_1 \cup A_2 \cup ... \cup A_n = \Omega \)
\( A_i \cap A_j = \empty \) for all \(i \ne j\)

Probability Axioms:

1. \(P(A) \ge 0 \) for all \( A \subset \Omega\)
2. \(P(\Omega)=1\)
3. For \( A_1,A_2,... \) disjoint events: \[ P(A_1 \cup A_2 \cup ...)=P(A_1)+P(A_2)+... \]

Properties:

\(P(\empty)=0\)
\(P(\overline{A})=1-P(A)\) for all \(A\subset \Omega \)
\( 0 \le P(A) \le 1 \) for all \(A\subset \Omega \)
For \(B\subseteq A\) we have \(P(B) \le P(A) \)

\( \Omega \) => Probability Space
\( P(\Omega) \) => Probability Measure


Addition rules:

Events \(A,B \subset \Omega\): \[ P(A\cup B)=P(A)+P(B)-P(A \cap B) \] Disjoint events, \(A,B \subset \Omega\): \[ P(A\cup B)=P(A)+P(B) \]

Classical Probability I:

For a Sample Space \(\Omega\), the classical probability measure is: \[ P(A)= \frac{\lvert A \rvert}{\lvert \Omega \rvert} \]

\( \lvert A \rvert \) => Cardinality of event A
\( \lvert \Omega \rvert \) => Cardinality of Sample Space

Conditional Probability:

Independence:

Two events are independent if: \[ P(A \cap B)=P(A)P(B) \]

Conditional probability:

\[ P(A \mid B)=\frac{P(A \cap B)}{P(B)} \] For two independent events: \[ P(A \mid B)=\frac{P(A)P(B)}{P(B)}=P(B) \]

Chain rule:

For events \( A_1,A_2,...,A_n \) we have: \[ P(A_1 \cap A_2 \cap ... \cap A_n)= \\ P(A_1)P(A_2 \mid A_1)P(A_3 \mid A_1 \cap A_2) ...P(A_n \mid A_1 \cap A_2 \cap ... \cap A_{n-1}) \]

Law of total probability:

If \( A_1,A_2,...,A_n \) are a partition of \(\Omega\) and B is another event: \[ P(B) = \sum_{i=1}^{n}P(A_i)P(B \mid A_i) \]

Bayes' theorem:

For events \(A\) and \(B\) with \( P(A), P(B) > 0 \) we have: \[ P(A \mid B)=\frac{P(A)P(B \mid A)}{P(B)} \]

Law of total probability mixed with Bayes':

If \( A_1,A_2,...,A_n \) are a partition of \(\Omega\) and B is another event: \[ P(A_i \mid B) = \frac{P(A_i)P(B \mid A_i)}{\sum_{j=1}^{n}P(A_j)P(B \mid A_j)} \]

CDF, Expectation, Variance

Below you can see definitions for: Discrete Random Variable, Cumulative Distribution Function, Expected Value, Variance, etc.


Discrete Random Variable:

Definition:
A random variable is a function \(X\) from \(\Omega\) to \(\real\), where every \(\omega\) is assigned a real number \(X(\omega)\). \[ P({\omega:X(\omega)=x}) \]


The set of values \(X\) can take is its range:
\[ Range(X) = {X(\omega): \omega \in \Omega} \]

Probability Mass Function:

Definition:
For a random variable \(X\) the probability mass function, PMF or \(p_X\) of \(X\) is: \[ p_X(X)=P(X=x), x \in Range(X) \]


For any set A, we have:
\[ P(X \in A) = \sum_{x \in A} P(X=x) = \sum_{x \in A} p(x) \]

Cumulative Distribution Function:

Definition:
For a random variable \(X\) the cumulative distribution function, CDF or \(F_X\) of \(X\) is: \[ F_X(X)=P(X \le x), x \in \real \]

Properties:

\[ \lim_{x \to -\infty}F_X(x)=0 \] \[ \lim_{x \to +\infty}F_X(x)=1 \] \[ 0 \le F_X(x) \le 1 \]


For any discrete random variable X, we have:
\[ F_X(x) = \sum_{y \le x} P(X=y) = \sum_{y \le x} p_X(y) \] \[ P(a \le X \le b)=F_X(b)-F_X(a) \]

Continuous random variable:

\[ (F_X(x))'_x=P_X(X=x) \]

Expectation:

Definition:
For a random variable \(X\), the expected value or the "average" is: \[ E(X)=\sum_{\omega \in \Omega}X(\omega) \cdot P(\omega) \]

Law of the unconscious statistician:

For a random variable \(X\), and another random variable \(Y=g(X)\): \[ E(Y)=Eg(X)=\sum_xg(x)p_X(x) \]


For \(p_X\) as PMF of \(X\), we have:
\[ E(X)=\sum_{\omega \in Range(X)}x \cdot p_X(x) \]

Linearity of expectation:

\[ E(aX) = a\cdot E(X) \] \[ E(X+b) = E(X)+b \]

Continuous random variable:

\[ E(X)= \int_{E(X)} x \cdot p(x)dx \]

Variance:

Definition:
For a random variable \(X\), and expected value \(E(X)=\mu \): \[ Var(X)=E[(X- \mu)^2] \]


\(Var(X)\) can also be calculated as:
\[ Var(X)=E(X^2) - \mu^2 \]

Classic Distributions

Here you can find formulas for: Uniform, Bernoulli, Binomial, Poisson and Geometric Distributions. PDF/PMF, Expectation and Std for each.


Discrete Uniform Distribution:

Probability Density Function:
\[ P(X=x) = \frac{1}{n} \]

\( n \) => number of equally likely outcomes

Expectation:
\[ E(X) = \frac{n+1}{2} \]

Variance:
\[ Var(X) = \frac{n^2-1}{12} \]

Bernoulli Distribution: \( X \sim B(p) \)

Probability Density Function:
\[ P(X=x) = \begin{cases} p & x=1 \\ 1-p & x=0 \end{cases}\]

\( p \) => parameter determining the probability of success

Expectation:
\[ E(X) = p \]

Variance:
\[ Var(X) = p(1-p) \]

Binomial Distribution: \( X \sim B(n,p) \)

Probability Density Function:
\[ P(X=k) = \binom{n}{k} p^k(1-p)^{n-k} \]

\( p \) => parameter determining the probability of success
\( n \) => number of trials
\( k \) => number of trials in study

Expectation:
\[ E(X) = n \cdot p \]

Variance:
\[ Var(X) = n \cdot p(1-p) \]

Poisson Distribution: \( X \sim P(\lambda) \)

Probability Density Function:
\[ P(X=k) = \frac{e^{-\lambda}\lambda^k}{k!} \]

\( \lambda \) => parameter determining the intensity
\( k \) => intensity in study

-> Poissonial Approximation

When a binomial distribution has a large number of trials, we can use the Poisson Distribution formula, substituting \(\lambda\) with binomial parameters, \(n\) and \(p\).
\(X \sim P(n,p) \)

Expectation:
\[ E(X) = \lambda \]

Variance:
\[ Var(X) = \lambda \]

Geometric Distribution: \( X \sim G(p) \)

Probability Density Function:
\[ P(X=k) = (1-p)^{k-1} \cdot p \]

\( p \) => parameter determining the probability of success
k => order of the first successful trial in study

Expectation:
\[ E(X) = \frac{1}{p} \]

Variance:
\[ Var(X) = \frac{(1-p)}{p^2} \]
